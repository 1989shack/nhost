# React-Apollo example

## Get started

1. Clone the repository

```sh
git clone https://github.com/nhost/nhost
cd nhost
```

2. Install dependencies

```sh
cd examples/react-apollo
pnpm install
```

3. Terminal 1: Start Nhost

```sh
nhost dev
```

4. Terminal 2: Start React App

```sh
pnpm run dev
```
