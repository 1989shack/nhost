# `useAuthLoading()`

:::caution Deprecated
When using both `useAuthLoading` and `useAuthenticated` together, their initial state will change
three times:

`(true, false)` -> `(false, false)` -> `(false, true)`

Use `useAuthenticationStatus` instead.
:::
