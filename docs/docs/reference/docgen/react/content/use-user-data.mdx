# `useUserData()`

User data

```js
const {
  id,
  email,
  displayName,
  avatarUrl,
  isAnonymous,
  locale,
  defaultRole,
  roles,
  metadata,
  createdAt
} = useUserData()
```

## Examples

```js
const {
  id,
  email,
  displayName,
  avatarUrl,
  isAnonymous,
  locale,
  defaultRole,
  roles,
  metadata,
  createdAt
} = useUserData()
```
