# `onAuthStateChanged()`

Use `onAuthStateChanged` to add a custom function that will trigger
whenever the state of the user changed. Ex from signed in to signed out or
vice versa.

## Parameters

---

**fn** <span className="optional-status">required</span> [`AuthChangedFunction`](/reference/javascript/auth/types/auth-changed-function)
