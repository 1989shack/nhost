# `NhostReactClientConstructorParams`

## Parameters

---

**backendUrl** <span className="optional-status">required</span> `string`

---

**Client** <span className="optional-status">optional</span> `AuthClient`

---

**autoLogin** <span className="optional-status">optional</span> `boolean`

When set to true, will parse the url on startup to check if it contains a refresh token to start the session with

---

**autoRefreshToken** <span className="optional-status">optional</span> `boolean`

When set to true, will automatically refresh token before it expires

---

**clientStorageSetter** <span className="optional-status">optional</span> `StorageSetter`

Define a way to set information about the refresh token and its exipration date

---

**clientStorageGetter** <span className="optional-status">optional</span> `StorageGetter`

Define a way to get information about the refresh token and its exipration date

---

**clientStorageType** <span className="optional-status">optional</span> `ClientStorageType`

---

**clientStorage** <span className="optional-status">optional</span> `ClientStorage`

---

**refreshIntervalTime** <span className="optional-status">optional</span> `number`

Time interval until token refreshes, in seconds

---
