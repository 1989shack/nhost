Hasura Auth runs in a container along side Postgres and Hasura.

## Nhost (recommended)

The recommended way to start using Hasura Auth is by using Nhost. You will get a complete backend ready in seconds with Hasura, authentication, storage and serverless functions.

Go to [Nhost](https://nhost.io) and start building your app now.

## Self hosting

```sh
git clone https://github.com/nhost/hasura-auth.git
cd hasura-auth
cp .env.example .env
docker-compose -f docker-compose-example.yaml up
```
